# Repository Structure

**Project:** Hypatia Knowledge Platform

**Document ID:** HYP-SPEC-010100

**Version:** 1.0 Founder Edition

**Status:** Official

**Classification:** Architecture

---

# Purpose

This document defines the official structure of the Hypatia Repository.

The Repository is the permanent scientific storage of all structured knowledge.

It is independent from:

* Programming language
* Database engine
* Website
* AI models

The Repository exists to preserve knowledge.

---

# Repository Philosophy

The Repository is organized according to knowledge responsibilities rather than software responsibilities.

Folders represent scientific concepts.

Not programming modules.

---

# Repository Root

```text
Hypatia/
```

Everything belongs under one root.

No parallel repositories.

No duplicated structures.

---

# Repository Ownership

Each folder has exactly one responsibility.

No folder may duplicate another folder.

---

# Scientific Rule

Knowledge lives only inside:

```text
repository/
```

Everything else either:

Consumes knowledge

Produces knowledge

Indexes knowledge

Reviews knowledge

Displays knowledge

Trains from knowledge

But only the Repository owns knowledge.
